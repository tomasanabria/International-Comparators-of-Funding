# The-Future-of-Bioethics

## Funder Cleaning Code

The funder cleaning and normalization workflow is stored in:

`scripts/funder_cleaning_script.py`

This script filters retained articles, extracts Scopus funding details, normalizes funder names, removes repeated acronym variants, deduplicates funders within each article, and generates funder mention counts, raw occurrence counts, and raw variant audit files.
